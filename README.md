# s95-kb

База знаний о волонтёрских ролях С95. Роли сопоставлены с [Sat_9am_5km](https://github.com/vol1ura/Sat_9am_5km) — платформой забегов «Суббота 9:00 5 км».

Начните с [общего описания волонтёрства](general/overview.md).

## Структура файлов

```
.
├── README.md
├── general/
│   └── overview.md                    # Общее описание волонтёрства и таблица ролей
└── roles/
    ├── director.md                    # director — Директор
    ├── marshal.md                     # marshal — Маршал
    ├── timer.md                       # timer — Секундомер
    ├── photograph.md                  # photograph — Фотограф
    ├── tokens.md                      # tokens — Раздача карточек позиций
    ├── scanner.md                     # scanner — Сканер
    ├── instructor.md                  # instructor — Инструктаж новичков
    ├── marking_maker.md               # marking_maker — Разметка трассы
    ├── event_closer.md                # event_closer — Замыкающий
    ├── marking_picker.md              # marking_picker — Сбор разметки
    ├── cards_sorter.md                # cards_sorter — Сортировка карточек
    ├── bike_leader.md                 # bike_leader — Ведущий велосипед
    ├── pacemaker.md                   # pacemaker — Пейсмейкер
    ├── results_handler.md             # results_handler — Обработка результатов
    ├── equipment_supplier.md          # equipment_supplier — Доставка оборудования
    ├── public_relations.md            # public_relations — Связи с общественностью
    ├── warm_up.md                     # warm_up — Проведение разминки
    ├── other.md                       # other — Разное
    ├── attendant.md                   # attendant — Сопровождающий
    ├── finish_maker.md                # finish_maker — Организация финиша
    ├── volunteer_coordinator.md       # volunteer_coordinator — Координатор волонтёров
    ├── food_maker.md                  # food_maker — Организация питания
    ├── videographer.md                # videographer — Видеограф
    ├── designer.md                    # designer — Дизайнер
    └── event_preparation.md           # event_preparation — Подготовка забега
```
