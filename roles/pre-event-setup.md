# Pre-event Set-up (подготовка до мероприятия)

Волонтёры Pre-event Set-up помогают [Run Director](run-director.md) подготовиться к мероприятию. Это может включать:

- размещение маркеров и/или указателей по трассе;
- подготовку зоны старта/финиша, включая финишный коридор;
- установку знаков и/или конусов, обозначающих маршрут.

**Категория:** дополнительная роль
