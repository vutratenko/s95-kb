# Секундомер (`timer`)

Секундомер отвечает за фиксацию финишного времени всех участников забега. В идеале мероприятия должны использовать два устройства хронометража, чтобы защититься от потери одного набора времён.

**Slug в [Sat_9am_5km](https://github.com/vol1ura/Sat_9am_5km):** `timer`
