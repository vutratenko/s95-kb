# Token Sorting (сортировка жетонов)

Волонтёры Token Sorting раскладывают финишные позиционные жетоны обратно по порядку на следующую неделю. Эта роль часто выполняется в кафе после parkrun и подчёркивает социальные преимущества волонтёрства.

**Категория:** дополнительная роль
