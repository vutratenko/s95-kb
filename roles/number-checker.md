# Number Checker (проверяющий номера)

Number Checker отвечает за фиксацию (но не исправление) расхождений, которые могут возникнуть между [хронометристом](timekeeper.md) и выданными номерами. Он тесно работает с хронометристом и волонтёрами [Finish Tokens](finish-tokens.md).

**Категория:** дополнительная роль
