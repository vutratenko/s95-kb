# Timekeeper (хронометрист)

Хронометрист отвечает за фиксацию финишного времени всех участников мероприятия (с помощью приложения Virtual Volunteer). В идеале мероприятия должны использовать два устройства хронометража, чтобы защититься от потери одного набора времён.

**Категория:** обязательная роль
