# Finish Tokens (финишные жетоны)

Волонтёр на роли Finish Tokens отвечает за раздачу позиционных жетонов всем участникам, работая в команде с [хронометристом](timekeeper.md). Ему может помогать волонтёр на роли [Finish Token Support](finish-token-support.md).

**Категория:** обязательная роль
