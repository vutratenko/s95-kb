# Report Writer (автор отчёта)

Волонтёр пишет отчёт о мероприятии выходного дня, который можно просмотреть на сайте мероприятия. Разные мероприятия подходят к написанию отчётов по-разному, и HQ не ожидает, что отчёт обязательно будет опубликован.

**Категория:** дополнительная роль
