# Разное (`other`)

Роль для волонтёрских задач, не попадающих в стандартный перечень. Конкретное поручение согласуется с [директором](director.md) или [координатором волонтёров](volunteer_coordinator.md).

В расписании можно указать короткий комментарий (4–40 символов) с описанием задачи.

**Slug в [Sat_9am_5km](https://github.com/vol1ura/Sat_9am_5km):** `other`
