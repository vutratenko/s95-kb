# Event Director (директор мероприятия)

Каждым мероприятием parkrun управляет один или два event director (директора мероприятия), которые отвечают за надлежащее проведение события, выстраивание отношений с владельцами территории и взаимодействие с parkrun HQ.

## Требования

- Возраст — 18 лет и старше.
- Один человек не может быть Event Director более чем для одного мероприятия одновременно.
- Для одного мероприятия допускается максимум два со-директора (Co-Event Directors).
