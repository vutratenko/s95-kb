# Run Director (директор забега)

Run Director несёт общую ответственность за безопасность мероприятия и контролирует ход события и работу на финише. Он должен постоянно находиться в зоне старта и/или финиша.

Run Director имеет окончательное право решать, следует ли отменить или перенести мероприятие, а также нужна ли корректировка трассы.

Run Director также приветствует всех участников parkrun на предстартовом брифинге и доносит все важные сообщения до начала события, отмечает вклад волонтёров, приветствует гостей и новичков и следит за тем, чтобы все понимали правила мероприятия и оставались в безопасности на протяжении всего события.

## Важные ограничения

- Лицо младше 18 лет может быть RD только при сопровождении взрослого RD на протяжении всего мероприятия.
- Run Director не должен участвовать в забеге или выполнять другую волонтёрскую роль во время проведения мероприятия.

**Категория:** обязательная роль
