# Event Day Course Check (проверка трассы в день мероприятия)

Event Day Course Check включает проверку того, что трасса безопасна для использования и на ней нет ничего, что может сделать её небезопасной для участников и волонтёров. Проверка проводится в день мероприятия как можно ближе к времени старта.

**Категория:** обязательная роль
