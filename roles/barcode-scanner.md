# Barcode Scanner (сканер штрихкодов)

Этот волонтёр сканирует персональный штрихкод участника (с помощью приложения Virtual Volunteer), а затем штрихкод на финишном позиционном жетоне, чтобы каждый участник, завершивший дистанцию, получил результат.

**Категория:** обязательная роль
