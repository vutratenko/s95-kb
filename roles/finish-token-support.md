# Finish Token Support (помощник по финишным жетонам)

Роль включает подготовку небольших стопок жетонов и передачу их волонтёру(ам) Finish Tokens, которые раздают их пересекающим финишную линию. Эта роль часто используется на крупных мероприятиях, где волонтёры Finish Tokens могут испытывать давление при своевременной раздаче позиционных жетонов.

**Категория:** дополнительная роль
