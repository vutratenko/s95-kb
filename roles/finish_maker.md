# Организация финиша

Этот волонтёр отвечает за зону финиша и следит, чтобы участники проходили через финишный коридор ровно, по порядку и без толкучки — это важно, ведь именно порядок прихода определяет позицию в протоколе. Он же помогает наладить работу [раздачи карточек](tokens.md) и [сканера](scanner.md), чтобы на финише всё шло как по нотам.
